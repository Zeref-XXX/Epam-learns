## ğŸ“š Content Overview

### JavaScript Practice

#### Basics
- **Variables and Data Types**: Learn about different ways to declare variables and work with various data types
- **Functions**: Practice function declarations, expressions, arrow functions, and more

#### Intermediate
- **Array Methods**: Master map, filter, reduce, and other array manipulation methods
- **Promises and Async/Await**: Understand asynchronous JavaScript patterns

#### Advanced
- **Closures and Higher Order Functions**: Explore advanced functional programming concepts, currying, composition, and memoization

### TypeScript Practice

#### Basics
- **Types and Interfaces**: Learn TypeScript type system, interfaces, and type annotations
- **Classes and Objects**: Practice object-oriented programming in TypeScript

#### Intermediate
- **Generics**: Master generic types for writing reusable code
- **Advanced Types**: Work with union types, intersection types, type guards, and more

#### Advanced
- **Decorators**: Practice metaprogramming with TypeScript decorators

### Running JavaScript Files
```bash

```

### Running TypeScript Files

1. Compile TypeScript to JavaScript:
```bash
npm run build
```

2. Run the compiled JavaScript:
```bash
node dist/typescript/basics/01-types-and-interfaces.js
```

Alternatively, use ts-node for direct execution:
```bash
npx ts-node typescript/basics/01-types-and-interfaces.ts
```

### Watch Mode
To automatically recompile TypeScript files on changes:
```bash
npm run build:watch
```
